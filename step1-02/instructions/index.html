Setup

Build the UX:
  • running yarn start from the root directory will build all libraries and run the entire studio UX
  • to speed up the development process, navigate to a specific app-module, run yarn lib in that directory, and yarn start to build only that app-module
  • check this troubleshooting page if you have any errors with the build
    • https://dev.azure.com/experimentation/Analysis%20and%20Experimentation/_wiki/wikis/AnE.wiki/5527/Troubleshooting
